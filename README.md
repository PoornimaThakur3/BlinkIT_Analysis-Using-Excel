# BlinkIT_Analysis-Using-Excel
This project provides a Real-Time Sales Analysis of BlinkIT using Excel. The analysis focuses on understanding sales performance, customer demand, and product category insights to help optimize business strategies and decision-making.
# Key Highlights
Product Insights - Analysis of sales by product categories and items.
Regional Analysis – Insights into Location-Wise sales performance.
Customer Behavior – Analysis of Purchase Patterns and Demand.
# Tools & Technologies
Microsoft Excel – For data cleaning, transformation, and visualization.
Pivot Tables & Charts – To summarize and analyze sales data.
